线性表

一、概念

线性表是最基本和最简单的一类数据结构，它表示的是线性结构。在线性结构中，数据元素之间存在着一对一的关系，其特点是数据元素之间按照某种规定存在一个顺序关系。

1.1线性表的定义

线性表：n个同类型数据元素的有限序列，记为：

L为表名；
i为数据元素a_i在线性表中的位序；
n为线性表的表除；n=0时成为空表。
数据元素之间的关系时：

除去第一元素a1外，都有唯一的前驱，除去最后元素an外，都有唯一的后继。

二 线性表的实现方式和基本操作

线性表是一种逻辑结构，在计算机上面表示可以使用顺序存储结构和链式存储结构来实现。

顺序存储结构是把逻辑上相邻的元素存储在物理位置相邻的存储单元上面，而链式存储结构是在数据元素种添加一些地址域或者辅助结构，用来存放数据元素之间的关系。
顺序存储结构在计算机中可以使用数组来实现，链式存储结构可以用链表来实现。

线性表的操作可以分为变动性操作和非变动性操作，而基于这2大类可以分为以下操作：

变动性操作：

  • 创建线性表（创建空表，创建带有元素的表）
  • 插入元素（根据位置插入，在尾部插入）
  • 修改元素值
  • 删除元素值（根据位置删除，根据元素值删除）

非变动性操作：

  • 获取线性表长度（顺序存储方式需要获取表最大容量）
  • 遍历线性表
  • 查找元素（根据位置查，根据元素值查）

这只是基本操作，有其他对元素的操作根据自己需求撰写算法。

三、 线性表顺序存储实现

在写一些插入函数的时候，需要返回一个状态来表示成功与否，这里写一个枚举类型来表示状态

enum Status { SUCCESS, FAIL, RANGE_ERROR, OVER_FLOW, EMPTY };

顺序存储实现需要数组实现，数据成员要有一个数组且要2个成员表示该线性表的长度和该表的最大容量。

3.1 类体结构：

template<class DataType>
class SeqList(){
private:
    DataType *data;    //数组
    int length;        //该线性表的长度
    int maxLength;    //该线性表的最大长度 
public:
    SeqList(int size);
    ~SeqList();
    int GetLength()const;
    int GetMaxLength()const;
    void display()const;
    Status Insert(DataType& elem);
    Status Insert(int i , DataType& elem);
    Status Delete(int i);
    Status Delete(DataType &elem);
    Status DeleteEqualValue(DataType &elem);
    Status GetData(int i , DataType &elem)const;
};

3.5 插入函数

3.5.1 插入elem元素到线性表尾部

//插入elem元素到线性表尾部
template<class DataType>
Status SeqList<DataType>::Insert(DataType &elem){
    //先判断线性表是不是满了，没满的话再进行下一步操作
    if(length==maxLength)
        return FAIL;
    else{
        //因为数组内排序是从0开始的，所以数组下标和长度是差1的
        //假设数组长度5，那数组下标则是0，1，2，3，4 
        //那么在第5个位置也就是length位置插入即可
        data[length]=elem;
        length++;
        return SUCCESS;
    }
}

3.5.2 插入elem元素到i位置

//插入elem元素到i位置
template<class DataType>
Status SeqList<DataType>::Insert(int i , DataType &elem){
    //先判断i位置是否合法，再判断表是否满了，以上条件都不满足则可以插入
    //满足 1<=i<=length
    if(1<=i || i<length)
        return RANGE_ERROR;//越界错误
    else if(length == maxLength)
        return FAIL;
    else{
        //插入元素，元素i位置以后的元素都往后挪
        //元素是从最后一个开始往后挪
        //删除和插入都是使用 j 和j-1这样的方式来表达元素前后关系，因为数组是从0开始的，循环次数都是要-1
        for(int j=length; j>=i;j--)
            data[j]=data[j-1];
        data[i-1]=elem;
        length++;
        return SUCCESS;
    }
}

3.6 删除函数

3.6.1 删除 i 位置的元素

//删除位置i的元素
template<class DataType>
Status SeqList<DataType>::Delete(int i){
    //判断i位置是否合法
    if( i<1 || i<length)
        return RANGE_ERROR;
    else{
        //删除元素是i位置以后的元素往前挪
        for(int j = i; j<=length;j++)
            data[j-1]=data[j];
        length--;
        return SUCCCESS;
    }
}

3.6.2 删除表内所有==elem的元素

//删除表内所有==elem的元素
emplate<class DataType>
Status SeqList<DataType>::DeleteEqualValue(DataType &elem){
    //遍历整个数组,判断里面的元素是否==elem，判断就删除
    for(int i=0;i<length;i++){
        //判断元素是否==elem
        if(data[i]==elem){
            for(int j = i;j<length;j++)
                data[j]=data[j+1];
            length--;
        }
    }
    return SUCCESS;
}

四、 线性表链式存储实现

链式存储要链表实现，首先先定义结点类型：

4.1 链表内结点类型

template
struct Node{
public:
    DataType *data;     //数据域
    Node *next//指针域
    Node(){
        data=0;
        next=NULL;
    }//结点的构造函数
}

4.2 链表类体结构

线性表类需要实现头节点和表长度，所以类体成员要有结点成员和int成员

template<class DataType>
class LinkList(){
private:
    Node<DataType> *head;        //头节点
    int length;                   //表长
public:
    LinkList();
    ~LinkList();
    void display();        //遍历链表
    int GetLength();    //获取该线性表的长度
    int LocateData(const DataType& data)const;        //定位元素位置
    Status GetData(int i, DataType& data)const;        //定位i位置，而后将i位置的元素返回给data
    Status InsertElemAtIndex(int i, const DataType& data);    //插入data元素到i位置
    Status Insert(const DataType& data);                    //将data元素插入到尾部
    Status DeleteElemAtIndex(int i, DataType& data);        //删除数据域为data的结点
}

4.2 构造函数

//构造函数
template<class DataType>
LinkList<DataType>::LinkList(){
    head = new Node<DataType>;
    head->data=0;
    head->next=NULL;
    length=0;
}

4.4.3 定位元素位置

//定位元素位置
template<class DataType>
int LinkList<DataType>::LocateData(const DataType &data)const{
    int count =1;
    Node<DataType> *p = head->next;
    while(p!=NULL){
        if(p->data==data)
            return count;
        count++;
        p=p->next;
    }
    if(p==NULL)
        return 0; //找不到该元素
}

4.5 插入函数

4.5.1 插入 data 元素到 i 位置

//插入data元素到i位置
template<class DataType>
Status LinkList<DataType>::InsertElemAtIndex(int i , const DataType &data){
    //判断i位置的合法性
    if( i<1 || i<=length)
        return RANGE_ERROR;
    else{
        //定位到i-1的位置，然后修改i-1的元素后继为i+1
        //然后释放i位置的结点
        Node<DataType> *p =head->next;
        for(int j = 1; j < i-1 ; j++)
            p=p->next;
        //构造新结点
        Node<DataType> *x= new Node<DataType>
        x->data = data;//数据域等于传进来的data
        x->next=p->next;//新节点的next== i-1元素的next
        p->next = x;    // i-1元素的next 变成新节点
        length++;
        return SUCCESS;
    }
}

4.5.2 插入data元素到尾部

//将data元素插入到尾部
template<class DataType>
Status LinkList<DataType>::Insert(const DataType &data){
    Node<DataType> *p = head;    //考虑到这个时候是空表，如果==head->next的话就出错了
    Node<DataType> *x = new Node<DataType>;
    x->data = data;
    x->next = NULL;
    //因为p指针是指向head的而不是head->next，所以循环变量从0开始
    //如果是空表的话，length==0，循环条件不成立也不会执行循环体语句还是指向head本身
    //如果不是空表，那么条件成立就会一直往下走
    for(int count = 0; cout < this-> length ; count++)
        p=p->next;
    p->next = x;
    length++
    return SUCCESS;
}

4.6.1 删除 i 位置的结点并将元素值返回到 data

//删除i位置的结点，并将元素值返回到data
template<class DataType>
Status LinkList<DataType>::DeleteElemAtIndex(int i,DataType &data){
   //判断i位置是否合法 1<=i<=length
    if (i < 1 || i >length)
        return RANGE_ERROR;
    else {
        Node<DataType>* p = head->next;
        for (int count = 1;count < i - 1;count++)
            p = p->next;
        Node<DataType>* q = p->next;  //指向i位置的结点
        p->next = p->next->next;    //i-1位置的next指针指向 i+1结点
        data = q->data;        //赋值给data带出去
        delete q;
        length--;
        return SUCCESS;
}
